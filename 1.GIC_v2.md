hwirq ↔ actual interrupt ID

#define GICD_CTLR          0x000
-  --> bit0=group 0 && bit_1 = group_1 enables the GIC control

#define GICD_ISENABLER     0x100  
- --> Interrupt Set-Enable Registers. Enable interrupts. (mask and unmask)
  - ISENABLER0 → IRQ 0-31
  - ISENABLER1 → IRQ 32-63

#define GICD_ICENABLER     0x180
- --> Interrupt Clear-Enable Registers. Disables interrupts. (mask and unmask)

#define GICD_ISPENDR       0x200
- --> Interrupt Set-Pending Registers. Marks interrupt as pending.
- Interrupt has occurred
- BUT not yet serviced

#define GICD_ISACTIVER     0x300 --> nterrupt Active Registers. Indicates interrupt is currently being serviced.

| State   | Meaning                   |
| ------- | ------------------------- |
| Pending | Waiting to be serviced    |
| Active  | CPU currently handling it |

Interrupt arrives
 → Pending=1

CPU ACKs interrupt
 → Active=1
 → Pending may clear

EOI written
 → Active clears
